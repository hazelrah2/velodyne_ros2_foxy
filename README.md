# velodyne_ros2_foxy
velodyne's driver for "ros-foxy-focal"

RUNNING
1. source /opt/velodyne/velodyne-lidar-driver-ros2/setup.bash
2. ros2 run lidar_driver_ros2 lidar_driver_ros2 --ros-args -p config_file:=/opt/velodyne/velodyne-lidar-driver/config/lidar_driver_velarray_m1600.cfg




INSTALLATION
1. Download files to machine
2. sudo dpkg -i ./velodyne-lidar-driver_2.4.0-22.05.11.1422_focal_amd64.deb
3. sudo dpkg -i ./ros-foxy-velodyne-lidar-driver_2.4.0-22.05.11.1422_focal_amd64.deb
